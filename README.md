# Scottish power rocks track react application
Scottish power rocks track react application.

## Instructions
 - $ git clone https://github.com/kibirb/sp-app-rock-track.git
 - $ cd sp-app-rock-track
 - $ npm install
 
## Dev
 - $ npm start
 - Browse to http://localhost:3000/
